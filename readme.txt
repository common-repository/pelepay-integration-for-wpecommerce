Integrate pelepay CC processing with your Wp-Ecommerce store

== Description ==

This plugin will allow you to integrate Pelepay credit card processing into your checkout process with Wp-Ecommerce store.

Management and initial setup through:

settings -> store -> payments -> pelepay

Insert your Pelepay account email is mandatory, all the rest comes preconfigured

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
